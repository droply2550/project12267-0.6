# 🛍️ Project 12267 - Web Store (Version 0.6)

เว็บไซต์นี้เป็นโปรเจกต์สำหรับแสดงสินค้าออนไลน์ (Static Website)
พัฒนาโดยใช้ HTML, CSS และ JavaScript และนำไป deploy บน GitHub Pages

---

## 🌐 Demo Website

🔗 https://droply2550.github.io/project12267-0.6/

---

## 📌 Features

* 🏠 หน้า Home แสดงภาพรวมเว็บไซต์
* 🛍️ หน้า Products สำหรับแสดงสินค้า
* 📞 หน้า Contact สำหรับติดต่อ
* 🎨 UI เรียบง่าย ใช้งานง่าย
* ⚡ โหลดเร็ว (Static Website)

---

## 🧱 Technologies Used

* HTML
* CSS
* JavaScript
* GitHub Pages (สำหรับ deploy เว็บไซต์)

---

## 📂 Project Structure

```
project12267-0.6/
│── index.html        # หน้า Home
│── products.html     # หน้าแสดงสินค้า
│── contact.html      # หน้าติดต่อ
│── css/              # ไฟล์ CSS
│── js/               # ไฟล์ JavaScript
│── images/           # รูปภาพสินค้า
```

## 🚀 How to Run (Local)

1. ดาวน์โหลดโปรเจกต์:

```
git clone https://github.com/droply2550/project12267-0.6.git
```

2. เข้าโฟลเดอร์โปรเจกต์:

```
cd project12267-0.6
```

3. เปิดไฟล์:

```
index.html
```
