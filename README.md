# Project 12267 
กิจกรรมภาคปฏิบัติ การทำสร้างเว็บไซต์ Responsive Design
เว็บไซต์นี้เป็นโปรเจกต์สำหรับแสดงสินค้าออนไลน์ (Static Website)
พัฒนาโดยใช้ HTML, CSS และ JavaScript และนำไป deploy บน GitHub Pages
